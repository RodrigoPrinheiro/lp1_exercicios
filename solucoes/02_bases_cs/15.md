###Pergunta
15 - Escreve um programa que mostre no ecrã todos os argumentos passados na
linha de comandos.

###Solução

##Solução 1
```cs
static void main(string[] args)
{
    foreach(string s in args)
        Console.WriteLine(s);
}
```
##Solução 2
```cs
static void main(string[] args)
{
    for(int i = 0; i < args.Lenght; i++)
        Console.WriteLine($"Argument {i}: {args[i]}");
}
```
